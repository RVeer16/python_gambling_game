# python_gambling_game
A simple intentionally rigged terminal based gambling game, Made for Fun

## How it works
- Start with 1000 coins
- Pick a number between 1 and 3
- Correct guess triples your coins
- Wrong guess halves your coins
- Game ends when you reach 10 coins or below
- Type 0 to quit anytime

## Why it's rigged
Win chance is 1/3 but loss chance is 2/3.
Long term you will always lose. Intentional design.

## How to run
python gambling_game.py
